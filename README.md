# Меню кафе Shaurmeals (menu Shaurmeals)

Одностроничное приложение «Меню кафе Shaurmeals» создано с использованием «Vue Router».

# Запуск

Просто откройте файл `index.html` в браузере - никаких зависимостей устанавливать не нужно.

# Структура проекта

```
Menu Shaurmeals/
├── index.html   # Разметка + логика
└── style.css    # Стили и внешний окрас приложения
```
